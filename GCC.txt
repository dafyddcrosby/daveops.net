====== GCC ======


| flag | desc |
| -S | get assembly code |
| -g | debug (for GDB) |
| -Wall | show warnings of questionable practices |
| -Wextra | warnings not covered by -Wall |
| -pedantic | warnings for strict ISO C |
| -std=c99 | use C99 standard |
| -Wformat-y2k | look for strftime code that uses double-digit year |
