GitHub - Enterprise
===================
:date: 2016-5-10
:modified: 2017-08-02

CLI
---
::

 # Set a message that's visible to everyone
 ghe-announce -s MESSAGE
 # Remove previously set message
 ghe-announce -u
 # Get webhook fails for a certain day
 ghe-webhook-logs -f -a YYYYMMDD
