====== CVS ======
:date: 2015-12-01
:tags: version control

===== Tagging an instance =====
--------------------
::

  cvs rtag -D "2010-1-28" tag_name module_name

===== Untagging an instance =====
--------------------
::

  cvs rtag -d tag_name module_name
