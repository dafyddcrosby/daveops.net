====== replication ======
@MySQL, @databases

===== Set the replica binlog coordinates =====

::

  CHANGE MASTER TO
	MASTER_LOG_FILE='mysqld-bin.123456',
	MASTER_LOG_POS=123456;


===== Check the binlog in a human readable way =====

::

  mysqlbinlog <binlog file>
