====== strace ======

| flag          | description                                      |
|:--------------|:-------------------------------------------------|
| -f            | trace child processes                            |
| -e trace=file | show only file operations                        |
| -i            | print instruction pointer at time of syscall     |
| -y            | print paths associated with file descriptor args |
