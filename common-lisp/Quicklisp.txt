====== Quicklisp ======
=========
:date: 2015-02-24

===== Load a system =====
--------------------
.. code-block:: lisp

   (ql:quickload "system-name")

===== Find a system =====
--------------------
.. code-block:: lisp

   (ql:system-apropos "term")

===== Load Quicklisp every time Lisp starts =====
--------------------
.. code-block:: lisp

   (ql:add-to-init-file)
