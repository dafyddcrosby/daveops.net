====== Linux - kernel modules ======

:tags: Linux

===== basic commands =====
::

  # list modules
  lsmod
  # get info about a module
  modinfo MODULE
  # load a module
  modprobe MODULE
  # load kernel module by filename
  insmod FILE
  # unload a module
  rmmod MODULE
