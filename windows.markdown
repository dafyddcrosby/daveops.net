# Windows (Microsoft)
Export certificates
-------------------

On Win7
	# From Start Menu
	certmgr.msc
	# Find cert, right-click, point to All Tasks, click Export


Recursive copy of files
-----------------------

On XP
	xcopy /E <src> <dest>


Hosts file
----------

c:\Windows\System32\Drivers\etc\hosts

Writing a batch file
--------------------

Save as FILENAME.bat
