====== Apple File System ======


@filesystems, @macOS

Replaces HFS+

OS X 10.12+

===== Features =====


* Container is base storage unit, generally 1:1 mapping to GPT
* 64-bit inaode numbers
* Nanosecond timestamp granularity
* Extensible block allocation
* Sparse files
* COW metadata
* Extended attributes
* TRIM operations
* Native encryption support, with multi-key encryption support
* SMB compatible
