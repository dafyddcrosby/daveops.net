====== Ruby - FileIO ======

@Ruby



===== Write to a file =====


.. code-block:: ruby

  fp = File.open(filename, mode)
  fp.write('bloop')
  fp.close

===== Read individual chars =====


.. code-block:: ruby

  fp = File.open(filename, mode)
  fp.each_char do |char|
	puts char
  end
  fp.close
